## 🚀 使用指南

### 快速开始

```bash
# 开发模式
npm run dev

# Electron 应用
npm run electron:dev

# 构建
npm run build

# 打包应用
npm run electron:build:win  # Windows
npm run electron:build:mac  # macOS
npm run electron:build:linux # Linux
```

### 核心工作流

1. **创建项目**
   - 点击"新建"或按 Ctrl+N
   - 配置项目参数
   - 选择模板

2. **设计页面**
   - 从左侧选择页面
   - 拖拽组件到画布
   - 在右侧面板编辑属性
   - 使用 AI 生成组件

3. **多页面管理**
   - 点击"添加页面"创建新页面
   - 右键页面可重命名/复制/删除
   - 点击页面切换编辑

4. **响应式预览**
   - 点击工具栏设备图标
   - 快速切换不同设备尺寸
   - 验证响应式效果

5. **保存和导出**
   - Ctrl+S 保存项目
   - Ctrl+E 导出 HTML
   - 在浏览器中打开预览
