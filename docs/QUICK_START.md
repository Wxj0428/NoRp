# NoRp UI 设计器 - 快速入门指南

## 🎯 5 分钟上手

### 第一步：启动应用
```bash
npm run electron:dev
```

### 第二步：使用 AI 生成页面

1. **打开 AI 面板**
   - 点击右下角 🤖 按钮

2. **选择快速模板**
   - 点击 "🔐 登录页" 或其他模板

3. **生成并插入**
   - 等待 AI 生成
   - 点击 "✅ 插入到画布"

### 第三步：编辑元素

1. **选择元素**
   - 在画布上点击任意元素

2. **修改样式**
   - 在右侧属性面板修改颜色、大小等

3. **调整位置**
   - 拖拽元素移动位置

### 第四步：保存项目

项目会自动保存，或使用 `Ctrl/Cmd + S` 手动保存。

```
5. 调整配色方案
6. 完成！
```

## 💡 实用技巧

### 技巧 1：精确描述需求

❌ 不好的描述：
```
做一个好看的产品页
```

✅ 好的描述：
```
生成一个产品详情页，包含：
- 左侧产品大图
- 右侧产品信息（标题、价格、购买按钮）
- 使用白色背景，橙色强调色
- 卡片式布局，阴影效果
```

### 技巧 2：迭代优化

第一次生成不满意？继续对话！

```
你：生成一个登录页
AI：[生成结果]

你：把背景改成紫色渐变
AI：[优化结果]

你：输入框添加圆角和阴影
AI：[继续优化]
```

### 技巧 3：使用页面说明

为每个页面记录设计思路：

1. 右键点击页面 > 页面说明
2. 记录：
   - 目标用户
   - 设计风格
   - 交互逻辑
   - 注意事项

### 技巧 4：善用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存 |
| `Ctrl/Cmd + Z` | 撤销 |
| `Delete` | 删除选中元素 |
| `Ctrl/Cmd + D` | 复制元素 |
| `Ctrl/Cmd + I` | 打开 AI 面板 |

## 🐛 常见问题

### Q: AI 生成太慢？
**A**:
- 使用快速模板（更快）
- 减少描述复杂度
- 检查网络连接

### Q: 如何修改已有元素？
**A**:
1. 点击选中元素
2. 右侧属性面板会显示所有可编辑属性
3. 修改后实时生效

### Q: 可以导入自己的 HTML 吗？
**A**:
当前版本不支持直接导入，但可以：
1. 复制 HTML 代码
2. 告诉 AI 帮你优化
3. 重新生成到画布

### Q: 项目文件在哪里？
**A**:
保存项目时选择的位置，默认 .norp 格式。

### Q: 如何导出代码？
**A**:
```
菜单 > 文件 > 导出
选择格式：HTML 单文件或 HTML+CSS 分离
```
