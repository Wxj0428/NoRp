## 应用概述

**NoRp UI 设计器** 是一款 AI 驱动的可视化 UI 设计工具，让设计网页变得简单高效。

### 核心特性

- 🤖 **AI 智能生成**：通过自然语言描述，AI 自动生成完整页面
- 🎨 **可视化编辑**：拖拽调整元素，实时预览效果
- 📱 **响应式设计**：自动适配不同屏幕尺寸
- 💾 **自动保存**：实时保存设计进度
- 🚀 **快速模板**：6 大类预设模板，一键生成
- 📄 **页面管理**：多页面项目管理，设计思路记录
- 🎯 **属性编辑**：精细控制每个元素的样式

### 适用场景

- 快速原型设计
- 登录/注册页面
- 管理后台界面
- 产品展示页面
- 营销落地页
- 数据可视化仪表板

## 快速开始

### 第一次使用

1. **启动应用**
   ```bash
   npm run electron:dev
   ```

2. **创建新项目**
   - 应用启动后会自动创建一个新项目
   - 项目包含一个默认页面

3. **使用 AI 生成页面**
   - 点击右下角的 🤖 按钮打开 AI 面板
   - 选择一个快速模板（如 🔐 登录页）
   - 点击"插入到画布"

4. **编辑元素**
   - 在画布上点击选择元素
   - 在右侧属性面板修改样式
   - 拖拽调整元素位置

5. **保存项目**
   - 项目会自动保存
   - 或使用菜单 `文件 > 保存项目`

### 5 分钟快速上手

| 时间 | 操作 |
|------|------|
| 0:00 | 启动应用，查看界面布局 |
| 0:30 | 打开 AI 面板，选择"登录页"模板 |
| 1:30 | 生成页面并插入到画布 |
| 2:30 | 选择元素，修改颜色和字体 |
| 3:30 | 拖拽调整元素位置 |
| 4:30 | 添加第二个页面 |
| 5:00 | 保存项目，导出 HTML |

## 核心功能

### 1. 项目管理

#### 创建新项目
```
菜单：文件 > 新建项目
快捷键：Ctrl/Cmd + N
```

#### 保存项目
```
菜单：文件 > 保存项目
快捷键：Ctrl/Cmd + S
说明：项目会自动保存，手动保存可确保数据安全
```

#### 打开项目
```
菜单：文件 > 打开项目
快捷键：Ctrl/Cmd + O
支持格式：.norp（项目文件）
```

### 2. 视图控制

#### 缩放画布
- `Ctrl/Cmd + +`：放大
- `Ctrl/Cmd + -`：缩小
- `Ctrl/Cmd + 0`：重置为 100%

#### 全屏预览
```
菜单：视图 > 全屏预览
快捷键：F11
```

#### 显示/隐藏面板
- `Ctrl/Cmd + 1`：显示/隐藏组件库
- `Ctrl/Cmd + 2`：显示/隐藏属性面板
- `Ctrl/Cmd + 3`：显示/隐藏页面列表

### 3. 编辑操作

#### 撤销/重做
```
撤销：Ctrl/Cmd + Z
重做：Ctrl/Cmd + Shift + Z
或使用工具栏按钮
```

#### 复制/粘贴
```
复制：Ctrl/Cmd + C
粘贴：Ctrl/Cmd + V
剪切：Ctrl/Cmd + X
```

#### 删除元素
```
快捷键：Delete 或 Backspace
右键菜单：删除元素
属性面板：删除按钮
```

## AI 设计助手

### 打开 AI 面板

点击右下角的 **🤖** 按钮打开 AI 设计助手。

### 快速模板（推荐新手）

AI 助手提供了 6 个快速模板，适合常见场景：

| 模板 | 图标 | 适用场景 | 特点 |
|------|------|----------|------|
| **登录页** | 🔐 | 用户登录、注册 | 渐变背景，卡片式布局 |
| **仪表板** | 📊 | 管理后台、数据分析 | 侧边栏 + 统计卡片 |
| **产品列表** | 🛍️ | 电商、产品展示 | 网格布局，产品卡片 |
| **表单页** | 📝 | 联系表单、反馈 | 清晰的表单验证 |
| **落地页** | 🎯 | 营销活动、产品推广 | CTA 引导，视觉冲击 |
| **设置页** | ⚙️ | 用户设置、偏好 | 分类清晰，交互友好 |

#### 使用快速模板

1. 打开 AI 面板
2. 在"快速模板"区域点击想要的模板
3. 输入框会自动填充详细的需求描述
4. 按回车或点击发送按钮
5. 等待 AI 生成页面
6. 点击"✅ 插入到画布"

### 自定义需求

如果你有特殊需求，可以自定义描述：

#### 描述技巧

✅ **好的描述**：
```

- 顶部用户头像和昵称
- 左侧导航菜单（个人信息、订单、收藏、设置）
```

4. **颜色风格**：指定颜色或风格（如蓝色系、暗色主题）
5. **交互要求**：特殊交互效果（如悬停动画）

## 页面管理

### 添加页面

**方法 1：使用页面列表**
1. 在左侧页面列表区域
2. 点击 **+** 按钮
3. 自动创建新页面

**方法 2：使用菜单**
```
菜单：页面 > 添加页面
快捷键：Ctrl/Cmd + Shift + N
```

### 切换页面

- 点击页面列表中的页面项
- 页面名称高亮显示当前页面

### 删除页面

**注意**：项目至少需要保留一个页面

1. 右键点击要删除的页面
2. 选择"删除页面"
3. 确认删除

### 重命名页面

**方法 1：双击重命名**
- 双击页面名称
- 输入新名称
- 按回车确认

**方法 2：右键菜单**
- 右键点击页面
- 选择"重命名"

### 复制页面

1. 右键点击页面
2. 选择"复制页面"
3. 自动创建副本

### 页面说明

为每个页面记录设计思路：

1. 右键点击页面，选择"页面说明"
2. 或点击页面右侧的 📄 图标
3. 在弹出的编辑器中记录：
   - 页面功能和目标用户
   - 设计思路和视觉风格
   - 交互流程和体验设计
   - 技术细节和注意事项

**示例说明**：
```
这是用户登录页面

💡 设计建议：
- 这是用户进入系统的第一印象
- 突出品牌形象，保持简洁
- 强调安全性（密码强度提示）
- 提供"记住我"和"忘记密码"功能

📝 设计思路：
- 使用紫色渐变背景营造科技感
- 卡片式登录框聚焦用户注意力
- 输入框焦点时有蓝色高亮
- 登录按钮使用渐变效果，hover 时有轻微放大
- 底部提供注册链接，引导新用户

⚠️ 注意事项：
- 表单验证：邮箱格式、密码长度
- 错误提示：登录失败时显示友好提示
- 加载状态：登录按钮显示加载动画
```

## 画布编辑

### 选择元素

**单击选择**：
- 点击元素即可选中
- 选中的元素会显示蓝色边框

**多选**：
- 按住 `Ctrl/Cmd` 点击多个元素
- 或使用拖拽框选

**取消选择**：
- 点击画布空白区域
- 按 `Esc` 键

### 移动元素

**方法 1：拖拽**
- 鼠标按住元素
- 拖动到新位置
- 释放鼠标

**方法 2：键盘微调**
- 选中元素后使用方向键
- `Shift + 方向键`：快速移动（10px）

**方法 3：属性面板**
- 在属性面板输入精确的 X、Y 坐标

### 调整大小

**方法 1：拖拽边缘**
- 鼠标悬停在元素边缘
- 出现调整大小光标时拖拽

**方法 2：属性面板**
- 在属性面板输入精确的宽度和高度

**方法 3：保持比例**
- 按住 `Shift` 拖拽角点
- 保持宽高比

### 画布工具

#### 标尺
```
菜单：视图 > 显示标尺
快捷键：Ctrl/Cmd + R
```

#### 网格
```
菜单：视图 > 显示网格
快捷键：Ctrl/Cmd + '
```

#### 辅助线
- 拖拽元素时自动显示对齐辅助线
- 帮助精确对齐其他元素

### 画布操作

#### 手型工具
- 按住空格键临时切换到抓手
- 拖拽平移画布

#### 缩放画布
- 鼠标滚轮：以鼠标位置为中心缩放
- `Ctrl/Cmd + 滚轮`：更精细的缩放控制

**方法 1：拖拽添加**

2. 拖拽到画布上

- 双击组件库中的组件
- 自动添加到画布中心

### 自定义组件

你可以将设计好的元素保存为自定义组件：

1. 选中要保存的元素
2. 右键点击"保存为组件"
3. 输入组件名称
4. 组件会添加到"我的组件"分类

## 项目导出

### 导出选项

```
菜单：文件 > 导出
快捷键：Ctrl/Cmd + E
```

#### 导出格式

**1. 纯 HTML（单文件）**
- 所有样式内联到 HTML
- 单个文件，可直接在浏览器打开
- 适合快速分享和预览

**2. HTML + CSS 分离**
- HTML 和 CSS 分开
- 结构清晰，便于维护
- 适合进一步开发

**3. Vue 组件**
- 导出为 .vue 文件
- 包含 template、script、style
- 适合 Vue 项目使用

**4. React 组件**
- 导出为 .jsx 文件
- 包含 JSX 和内联样式
- 适合 React 项目使用

### 导出设置

#### 基础设置
- ✅ 压缩 HTML/CSS/JS
- ✅ 包含响应式 meta 标签
- ✅ 添加注释说明

#### 高级设置
- ✅ 添加 CDN 依赖

## 常见问题

### Q1: AI 生成的页面如何修改？

**A**: 有两种方式：

1. **可视化编辑**：
   - 在画布上选择元素
   - 在属性面板修改样式
   - 实时预览效果

2. **继续与 AI 对话**：
   - 描述想要的修改
   - AI 会重新生成优化后的页面
   - 选择替换或合并到画布

### Q2: 如何自定义 AI 生成页面的风格？

**A**: 在描述需求时明确指定：

```
生成一个登录页：
- 背景：深色渐变（从深蓝到紫色）
- 输入框：半透明玻璃效果
- 按钮：霓虹发光效果
- 整体风格：赛博朋克、科技感
```

### Q3: 生成的代码质量如何？

**A**: AI 生成的代码特点：

- ✅ 语义化 HTML5
- ✅ 内联 CSS（无外部依赖）
- ✅ 响应式设计
- ✅ 现代化设计风格
- ✅ 清晰的代码结构
- ✅ 可直接使用或二次开发

### Q4: 项目文件存储在哪里？

**A**:

- **开发模式**：项目保存在应用内存中
- **保存后**：.norp 文件存储在用户选择的位置
- **自动保存**：临时数据保存在应用缓存

建议定期手动保存项目。

### Q5: 如何导出用于生产环境？

**A**: 导出时选择：

1. **格式**：HTML + CSS 分离
2. **设置**：
   - ✅ 压缩代码
   - ✅ 包含响应式 meta
   - ✅ 添加 CDN 依赖（如果需要）
3. 导出后进行测试和优化

### Q6: 支持哪些浏览器？

**A**:

- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ⚠️ IE11（有限支持）

建议使用现代浏览器以获得最佳体验。

### Q7: AI 助手需要 API Key 吗？

**A**: 取决于使用的服务：

- **Claude**：需要 Anthropic API Key
- **OpenAI**：需要 OpenAI API Key
- **本地模型**：需要安装 Ollama 或 LM Studio

### Q8: 生成速度慢怎么办？

**A**: 优化建议：

1. 使用更快的模型（如 GPT-3.5）
2. 减少需求的复杂度
3. 使用快速模板而非自定义描述
4. 检查网络连接

### Q9: 如何撤销多个操作？

**A**:

- 连续按 `Ctrl/Cmd + Z` 撤销多个操作
- 使用 `Ctrl/Cmd + Shift + Z` 重做
- 查看历史记录面板（即将推出）

### Q10: 页面说明有什么用？

**A**: 页面说明的作用：

- 📝 记录设计思路和决策
- 👥 团队协作时共享信息
- 🔄 以后维护时了解设计意图
- 📚 作为设计文档的一部分

### Q11: 可以导入已有的 HTML 吗？

**A**: 目前版本不支持直接导入，但你可以：

1. 复制 HTML 代码
2. 告诉 AI："帮我优化这个页面：[粘贴代码]"
3. AI 会帮你重新设计并添加到画布

### Q12: 元素如何精确定位？

**A**: 多种方式：

1. **拖拽**：大致定位
2. **属性面板**：输入精确坐标
3. **键盘微调**：方向键精确调整
4. **辅助线**：自动对齐其他元素

### Q13: 如何创建响应式设计？

**A**:

- AI 生成的页面默认是响应式的
- 手动编辑时使用百分比宽度
- 在属性面板设置断点样式
- 预览模式查看不同屏幕效果

### Q14: 项目可以多人协作吗？

**A**: 当前版本不支持实时协作，但可以：

1. 保存 .norp 项目文件
2. 通过文件共享或版本控制协作
3. 合并时手动整合修改

实时协作功能正在开发中。

```
第 1 次：生成一个仪表板
```

### 设计规范

#### 配色建议

**单色系搭配**：
- 主色：一种颜色的不同深浅
- 适合：极简、专业风格

**类比色搭配**：
- 色环上相邻的颜色
- 适合：和谐、自然的风格

**互补色搭配**：
- 色环上相对的颜色
- 适合：活力、对比强烈的风格

#### 间距系统

使用 8px 基础单位：
- 4px（0.5x）：极小间距
- 8px（1x）：小间距
- 16px（2x）：常规间距
- 24px（3x）：中等间距
- 32px（4x）：大间距

#### 字体大小

```
h1: 32px / 2.5rem
h2: 28px / 2rem
h3: 24px / 1.5rem
body: 16px / 1rem
small: 14px / 0.875rem
```
