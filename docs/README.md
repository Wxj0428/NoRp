## 🎓 学习路径

### 初学者（第 1 天）
1. ✅ 阅读 [快速入门指南](QUICK_START.md)
2. ✅ 跟随教程创建第一个页面
3. ✅ 尝试所有快速模板

### 进阶用户（第 2-3 天）
1. ✅ 阅读 [用户操作手册](USER_MANUAL.md) 的核心功能章节
2. ✅ 学习自定义 AI 需求
3. ✅ 掌握属性面板和画布编辑

### 高级用户（第 4-7 天）
1. ✅ 完整阅读 [用户操作手册](USER_MANUAL.md)
2. ✅ 学习 [最佳实践](USER_MANUAL.md#最佳实践)
3. ✅ 了解响应式设计
4. ✅ 掌握项目导出和部署

### 专家用户（持续）
1. ✅ 阅读 [术语表](GLOSSARY.md#-术语表) 深入理解技术细节
2. ✅ 学习故障排查和调试
3. ✅ 参与社区贡献

## 💡 文档使用技巧

### 阅读技巧

1. **按需阅读**：根据当前需求查找对应章节
2. **边做边学**：打开应用跟随文档操作
3. **标记重点**：使用书签或笔记记录重要内容
4. **搜索优先**：遇到问题先搜索文档

### 符号说明

| 符号 | 含义 |
|------|------|
| ⭐ | 推荐阅读 |
| ⚠️ | 注意事项 |
| ✅ | 推荐做法 |
| ❌ | 避免的做法 |
| 💡 | 实用技巧 |
| 🔧 | 需要配置 |
| 📖 | 延伸阅读 |

### 代码块
